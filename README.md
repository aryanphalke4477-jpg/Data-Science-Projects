## Projects Included

### 1. Used Cars Price Prediction

This project uses machine learning to predict the selling price of used cars based on features such as car age, fuel type, kilometers driven, and transmission type. It demonstrates regression techniques, data preprocessing, and model evaluation using Python libraries like Pandas, NumPy, and Scikit-learn.

### 2. Movie Recommendation System

This project builds a movie recommendation system using content-based filtering. It analyzes movie features such as genres, keywords, cast, and director to recommend similar movies to users. Cosine similarity is used to find relationships between movies.
